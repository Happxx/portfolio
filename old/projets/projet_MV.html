Installation d’un poste de travail pour le développement

Thème : Installer et configurer un poste de travail

Date : 2022

L'objectif du projet est de savoir comment installer et configurer un poste de travail sous Linux avec Debian 11 et KDE/Plasma sur une machine virtuelle, ainsi que d'installer les logiciels propices au développement tels qu'un JDK (Java Development Kit), Git et IntelliJ IDEA Community.

J'ai donc réalisé un diagramme pour illustrer les étapes nécessaires pour accomplir cette tâche.
